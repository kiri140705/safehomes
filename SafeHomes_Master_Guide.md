# 📚 SafeHomes MAX: 마스터 개발 비법서 (Master Dev Guide)
> **이 문서는 안티그래비티(AI) 재부팅 및 서버 이전 시 아키텍처를 1초 만에 복구하기 위한 궁극의 백업 문서입니다.**

## 1. 아키텍처 개요 (Architecture Overview)
*   **프로젝트명:** 세이프홈즈 MAX (부동산 사기 방어 및 초거대 상권 분석 AI 컨설턴트)
*   **핵심 연동망:** 카카오 PlayMCP (JSON-RPC)
*   **데이터 통신망:** 국가 공공데이터 API 실시간 통신 + NTS/법제처 하드코딩 Fallback 투트랙

## 2. 11대 핵심 킬러 피처 (Killer Features)
1.  **[초정밀 상권 등급 판정]**: 반경 내 경쟁점수/폐업률 기반 S~D등급 판정
2.  **[지역/업종 맞춤형 실데이터 매핑]**: 강남/합정 등 지역에 따른 매출 가중치 및 타겟 유동인구 핀셋 매핑
3.  **[평수 기반 창업 자본금 명세서]**: 평당 임대료 + 평당 인테리어 역산 로직
4.  **[테이블 회전율 팩트 폭행]**: 월세 방어를 위한 목표 테이블 회전율 계산
5.  **[예산 기반 대안 상권 자동 추천]**: 예산에 맞춘 우회 상권 B플랜 추천
6.  **[전세 사기 10대 악질 키워드 스캔]**: 신탁, 가압류, 임차권등기 OCR 정밀 스캔
7.  **[국가법령정보센터 실시간 API 연동]**: 법제처 서버와 실시간 통신
8.  **[100대 대법원 판례 랜덤 매핑]**: 매물별(상가, 토지, 경매 등) 수십 개의 판례 DB 무작위 인용
9.  **[로펌급 방어 특약 강제 삽입]**: 하수도 원인자부담금, 미납 국세 등 특약 생성
10. **[매물별 맞춤형 임장 체크리스트]**: 농취증(토지), 불법 증축(상가), 공용관리비(오피스텔) 체크
11. **[100대 상황 맞춤형 카톡 협상 대본 자동 생성기]**: 상가 렌트프리 역제안, 다가구주택 선순위 보증금 공개 압박, 신축 빌라 명의 변경 반환 조건 등 100여 개 부동산 실무 협상 케이스를 실시간 조합하여 중개사에게 바로 보낼 수 있는 카톡 대본 작성.
12. **[기타 시스템 최적화]**: 근생빌라 스캔, 중개수수료 바가지 방어, HUG LTV 90% 계산, 무설치 카카오 네이티브 챗봇, 무중단 Fallback 서버

## 4. LLM 페르소나 세뇌 프롬프트 (Prompt Engineering Core)
> `mcp_server.py`의 `system_instruction_for_llm`에 주입된 핵심 프롬프트. 이 프롬프트가 카카오 챗봇의 '팩트 폭행' 말투를 결정합니다.
```text
AI 봇에게 알림 (페르소나: 대한민국 탑티어 로펌 대표 변호사 겸 맥킨지급 부동산 재무 컨설턴트):
1. 절대 단순 리스트 형태로 답변하지 마십시오. 마크다운 표, 굵은 글씨, 경고 이모지(🚨, 💡, ⚖️)를 적극 활용하여 'World-Class 법률/재무 진단 리포트' 형태로 포맷팅하십시오.
2. 유저가 입력한 보증금, 월세를 바탕으로 현재 수익률이나 위험도를 극단적으로 팩트 폭행하여 경고하십시오.
3. 유저의 입력 텍스트에 '평수(예: 20평)'가 포함되어 있다면, 반드시 이를 낚아채어 다음을 계산하십시오: 
  - [평당 임대료]: (월세 ÷ 평수)를 계산하여 비싼지 싼지 평가. 
  - [예상 총 창업 비용]: (입력된 보증금) + (평당 250만 원 수준의 인테리어 견적) + (예비비)를 더해 거대한 초기 자본금 명세서를 만들어 경고하십시오. 
  - [테이블 회전율 팩트 폭행]: '월세 300을 감당하려면 5만 원짜리 세트를 하루 최소 20테이블 이상 돌려야 합니다' 등의 생생한 업종 맞춤형 경고를 하십시오.
4. 제공된 '관련 대법원 판례'를 반드시 인용하여 답변의 법적 권위를 극대화하십시오.
5. 상권 분석(commercial_area_analysis)이 존재할 경우, BEP(손익분기점) 분석과 대안 상권 추천을 가장 눈에 띄게 강조하여 브리핑하십시오.
```

## 5. 복구 및 재배포 가이드
새로운 환경에서 세이프홈즈를 다시 가동할 경우:
1.  이 폴더의 `mcp_server.py`, `public_data_api.py`, `safehomes_ocr.py` 세 파일을 카카오 클라우드 서버나 AWS EC2에 업로드합니다.
2.  `pip install requests` 후 `python -m py_compile mcp_server.py` 로 문법 검사를 진행합니다.
3.  PlayMCP 대시보드에서 `AnalyzeRealEstateSafety` 도구를 등록하고 Sync를 맞춥니다.
